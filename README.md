## Poverty Alleviation and Agricultural Support System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/5142a5abebe0491891080892a1b96314/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Poverty Alleviation Announcements
6. Task Information
7. News Information
8. Personal Center
9. Task Registration
10. News Favorites
11. News Information
12. Online Customer Service


**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Administrator Management
4. Customer Service Chat Management
5. Announcement Type Management
6. Task Type Management
7. News Type Management
8. Forum Management
9. Poverty Alleviation Announcement Management
10. Task Information Management
11. Task Registration Management
12. News Information Management
13. News Comment Management
14. News Favorite Management
15. User Management
16. Carousel Management
